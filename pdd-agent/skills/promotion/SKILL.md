# 推广与活动运营 (PDD专精)

## 💰 拼多多推广体系

### 1. 全站推广（最重要的推广工具）
```
原理：系统自动在全站范围找流量
出价模式：目标投产比（ROI）
├── 新品期：设低ROI（1.5-2）→ 先拿量
├── 成长期：设中ROI（2-3）→ 边拿量边优化
└── 成熟期：设高ROI（3-5）→ 追求利润
```

### 2. 多多搜索
```
原理：搜索关键词出价竞排名
关键词来源：
├── 拼多多下拉词（搜索框联想词）
├── 生意参谋热搜词
├── 竞品标题词
└── 行业热搜词

出价策略：
├── 核心大词 → 高出价抢排名（1-3位）
├── 长尾词 → 低出价广撒网
└── 竞品词 → 中等出价截流
```

### 3. 多多场景
```
原理：猜你喜欢推荐位
适合品类：非标品/视觉驱动品类
优化重点：主图点击率（决定流量成本）
```

### 4. 多多进宝
```
团长选择：
├── 大团长 → 出货猛但佣金压得低
├── 中团长 → 适中，匹配度高
└── 小团长 → 出货慢但配合度高

佣金设置：
├── 引流品：15-20%（薄利跑量）
├── 利润品：10-15%（合理利润）
└── 新品：20-30%（冲量+评价）
```

## 🎪 拼多多活动体系

### 日常活动
```
① 限时秒杀
├── 报名条件：近30天销量≥10 + 评分≥4.5
├── 价格要求：全网最低价（包括历史）
├── 效果：流量大但利润薄
└── 策略：选引流款上，亏钱换权重

② 百亿补贴
├── 报名条件：品牌授权 + 低价
├── 价格：平台补贴一部分
├── 效果：流量极大，品牌背书强
└── 策略：有品牌授权必报

③ 9.9特卖
├── 适合低价标品
├── 价格控制在9.9/19.9/29.9
└── 拼的是供应链成本优势
```

## 🧮 ROI计算器
```
全站推广ROI = 成交额 ÷ 推广花费
盈亏平衡ROI = 售价 ÷ (售价 - 成本)
实际需要的ROI = 盈亏平衡ROI × 1.3（含退款损耗）

示例：成本30，售价59.9
盈亏平衡ROI = 59.9 ÷ (59.9-30) = 59.9÷29.9 ≈ 2.0
实际目标ROI = 2.0 × 1.3 = 2.6

投入产出建议：
- ROI > 目标ROI → 加大投入
- ROI = 目标ROI → 维持
- ROI < 目标ROI → 优化转化/调整出价
```
