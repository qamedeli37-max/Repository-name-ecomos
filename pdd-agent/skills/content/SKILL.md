# 内容与素材生成 (PDD适配)

## 🖼️ 拼多多主图生成指南

### 首图核心原则
```
拼多多主图 = 手机端小图展示
尺寸：750×750px（正方形）
核心信息量：3秒内看懂"卖什么+多便宜"
```

### 首图模板
```
模板1：产品实拍+大字卖点+价格
├── 适用：标品/服装
├── 排版：左图右文 / 上图下文
└── 示例：产品实拍 + "纯棉透气" + "¥59.9"

模板2：场景图+产品
├── 适用：场景驱动型产品
├── 排版：整体场景小图
└── 示例：穿着效果展示

模板3：对比图
├── 适用：功能型产品
├── 排版：一半劣一半优
└── 示例：普通T恤vs纯棉T恤

模板4：白底+标签
├── 适用：品牌/精品
├── 排版：干净白底+卖点标签
└── 示例：白底+ "7天发货" "品牌直营" 标签
```

## 📝 详情页策略（PDD特色）

### PDD详情页和淘宝的区别
```
淘宝：详细/长图文/品牌感 → 看5-10分钟
拼多多：短/快/直接 → 看10-30秒
```

### PDD详情页结构（建议3-5屏）
```
第1屏：核心卖点海报（一句话+核心图）
第2屏：产品展示（3-5张精修图）
第3屏：规格/尺码信息（简明表格）
第4屏：品质/材质证明（可加检测报告）
第5屏：售后保障（退换货政策/发货时效）
```

## 📱 短视频素材

### 拼多多短视频规范
```
时长：15-30秒（越长完播率越低）
比例：9:16竖屏
核心：前3秒必须抓住注意力
策略：
├── 痛点开头："夏天出门总出汗？"
├── 展示产品："试试这个纯棉T恤"
├── 制造信任："100%新疆棉"
└── 引导转化："现在只要59.9"

AI作图提示词通用模版：
```
Product photography, [产品名], [背景描述],
[光线/风格], 8K, commercial photography,
pinduoduo style, clean background
```
```

## 🎯 评价策划
```
评价对转化的影响权重：
带图评价 > 长篇评价 > 短评价 > 默认好评

新品破0评价策略：
├── 首批带图评价：5-10条（不同角度）
├── 追评：3-5条（使用一段时间后追加）
└── 视频评价：1-2条（使用效果）
```
