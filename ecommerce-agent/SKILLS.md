# SKILLS.md — 技能索引

本目录包含以下技能模块，供 Codex 加载使用：

| 技能 | 目录 | 说明 |
|------|------|------|
| 🏷 商品上架 | skills/product-lister/ | 标题/SKU/详情/主图生成 |
| 🎨 内容生成 | skills/content-generator/ | 图片提示词/视频脚本/小红书 |
| 📊 数据分析 | skills/data-analyzer/ | 销售/库存/竞品分析 |
| 📱 社媒运营 | skills/social-media/ | 抖音/小红书/朋友圈文案 |

## 快速使用

在 Codex 工作区根目录启动后，直接跟它说：

```
"帮我上架这5个商品到拼多多"
"分析昨天的销售数据"
"给这批商品生成抖音脚本"
"写个小红书种草文案"
```
