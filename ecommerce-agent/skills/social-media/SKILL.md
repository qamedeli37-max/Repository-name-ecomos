# 社媒运营助手 (Social Media)

## 功能说明
生成各平台（抖音/小红书/视频号）的社媒内容，包括文案、话题标签、发布策略。

## 输出类型

### 1. 抖音短视频
- 脚本结构：前3秒钩子 → 痛点/需求 → 产品展示 → 引导
- 口播文案
- 字幕建议
- BGM/音效建议

### 2. 小红书笔记
- 爆款标题（吸引点击）
- 正文（个人体验+产品安利）
- 图片方案（9宫格排版建议）
- 标签策略（大词+精准词+长尾词）

### 3. 朋友圈/社群文案
- 短文案
- 适合私域转化的表达方式

## 输出示例
```markdown
## 抖音脚本：纯棉T恤

• 类型：好物分享
• 时长：30s
• BGM：轻快节奏

【0-3s】特写：出汗湿透的衣服
口播：夏天出门5分钟就这样了？

【3-10s】切到产品
口播：直到我换了这个纯棉T恤...
字幕：100%新疆棉

【10-20s】展示细节：透气测试/做工/版型
口播：透气到能看见光，做工比专柜还细

【20-25s】上身展示
口播：关键是才59块9

【25-30s】口播：左下角自己看
字幕：限时福利

#好物分享 #夏日穿搭 #纯棉T恤 #平价好物
```
