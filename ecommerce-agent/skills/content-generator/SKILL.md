# 批量化内容生成 (Content Generator)

## 功能说明
根据商品清单批量生成产品图片提示词、短视频脚本、营销文案、直播话术。

## 核心思路（参考 @阳雨Ai）
> 飞书多维表 + 工作流监听 → Codex 自动处理 → 结果写回 → 推送通知

参考阳雨Ai的 "Codex+飞书+工作流，日产1000+视频" 模式：
1. 在飞书多维表中提交任务（商品名称/类型/数量）
2. Codex 监听新记录，自动处理
3. 逐条生成文案/脚本/图片描述
4. 结果写回多维表或保存到 workspace

## 输入
- 商品 CS V 清单
- 或者直接在对话中描述

## 生成类型

### 1. 产品主图提示词（用于 AI 作图）
```
纯棉短袖T恤主图，白色背景，模特上身效果，自然光，高清商业摄影，
极简风格，突出面料质感，构图居中，8K
```

### 2. 短视频脚本（15s/30s/60s）
```markdown
## 15s 脚本
[0-3s] 产品特写 + 痛点文字：夏天穿什么？
[3-8s] 模特展示上身效果，口播卖点
[8-12s] 细节特写（面料/做工）
[12-15s] 价格+引导购买

## 30s 脚本
...
```

### 3. 详情页文案
按 痛点→解决方案→产品亮点→材质工艺→尺码建议→售后保障 结构输出

### 4. 直播话术
- 开场吸引停留
- 产品介绍（FAB法则）
- 价格锚点 + 逼单话术

### 5. 小红书种草文案
- 标题党风格标题
- 正文：个人体验+产品亮点+购买建议
- Tag 建议

## 批量模式
读取 workspace/products/products.csv，逐条生成并保存到：
- workspace/content/images/prompts/  — 图片提示词
- workspace/content/videos/scripts/   — 视频脚本
