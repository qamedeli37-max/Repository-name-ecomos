# 商品上架助手 (Product Lister)

## 功能说明
根据商品信息自动生成各平台（淘宝/拼多多/抖音小店）的上架内容，包括标题优化、描述撰写、SKU 方案、主图建议。

## 触发方式
- 直接描述商品信息触发
- 读取 workspace/products/ 下的 CSV 文件批量处理
- 配合飞书多维表自动触发

## 输入格式
```csv
商品名称,品类,卖点,成本价,目标售价,平台,材质,重量
纯棉短袖T恤,服装,透气/纯棉/百搭,25,59.9,拼多多,100%棉,0.2kg
```

## 输出内容
对每个商品生成：
1. **标题** — 3个版本（精准型/引流型/长尾词型）
2. **描述文案** — 详情页核心卖点文案（5-8条）
3. **SKU 方案** — 规格组合 + 价格建议
4. **主图建议** — 白底图/场景图/卖点图方案
5. **关键词** — 搜索关键词建议（10-15个）

## 平台规则

### 淘宝
- 标题上限30字
- 主图：5张（首图白底，其余卖点/细节/场景/尺码）
- 详情页：750px宽，建议 8-12 屏
- SKU：颜色+尺码是最常见组合

### 拼多多
- 标题上限30字（重视搜索权重词）
- 主图：首图决定点击率，需突出价格/卖点
- SKU：最低价SKU决定引流效果
- 详情页：移动端优先，大图少字

### 抖音小店
- 标题20字内
- 主图：视频封面感，高清产品+文字
- 重视短视频素材配合
- SKU：简单清晰，避免太多选择

## 输出文件
- workspace/products/listings/YYYY-MM-DD_商品名.md
- 支持批量产出到目录
