# 运营数据分析 (Data Analyzer)

## 功能说明
分析店铺运营数据，生成可读的日报/周报/月报，辅助决策。

## 数据来源
- workspace/products/sales_*.csv — 销售数据
- workspace/products/inventory_*.csv — 库存数据
- 用户直接提供的数据

## 分析维度

### 1. 销售分析
- 日/周/月销售趋势
- 爆款识别（销量TOP10）
- 滞销品识别（销量末尾+超过30天无动销）
- 退款率分析
- 客单价分布

### 2. 库存分析
- 库存周转率
- 缺货预警
- 积压库存提醒
- 安全库存建议

### 3. 竞品分析
- 价格对比
- 卖点差异
- 评价关键词提取

### 4. ROI 分析
- 推广花费 vs 成交额
- 单品利润率
- 广告ROI

## 输出格式
Markdown 报告，包含：
- 关键数据一览（数字摘要）
- 趋势图表（文字描述 + ASCII 示意）
- 异常预警
- 运营建议

## 输出文件
- workspace/reports/daily/YYYY-MM-DD.md
- workspace/reports/weekly/YYYY-WW.md
- 支持导出 CSV 汇总
