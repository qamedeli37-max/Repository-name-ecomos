### Task 27-29: Self-Awareness + Profiles + Cognition

#### Task 27: Meta-State
- Added MetaStateManager: current_strategy, strategy_history, performance metrics
- Critic → Meta-Critic: evaluates strategy effectiveness + execution efficiency
- Planner accepts meta_state input for self-reflection
- Files: `app/agent/meta.py`

#### Task 28: Behavioral Profiles
- Added ProfileManager with 5 profiles: balanced, efficient_executor, safe_executor, aggressive, learning
- Each profile: planner_style, critic_threshold, max_replans, retry_enabled, auto_fix
- Meta-Critic can switch profiles based on performance
- Planner/Executor accept profile parameter
- Files: `app/agent/profile.py`

#### Task 29: Cognition Levels
- Added CognitionLevel: shallow (reasoning:2), medium (reasoning:5), deep (reasoning:8)
- Profile → Cognition mapping: efficient→shallow, balanced→medium, safe/learning→deep
- Planner uses cognition level to adjust reasoning depth
- Critic validates cognition-plan match (shallow+long plan = penalty)
- Files: `app/agent/cognition.py`

## Current Architecture (Task 29)

```
User Input
    ↓
Agent._parse_goal() → _detect_goal_type()
    ↓
ProfileManager.get_current() → profile
    ↓
get_cognition_level(profile.id) → cognition
    ↓
StrategyRegistry.get_current() → strategy
    ↓
PlannerAgent.plan(goal, tools, memory, strategy, meta_state, profile, cognition)
    ↓
CriticAgent.evaluate(goal, plans, memory, strategy, meta_state, profile, cognition)
    ↓
if suggested_strategy → StrategyRegistry.switch_to()
if suggested_profile → ProfileManager.switch_to() + update cognition
    ↓
ExecutorAgent.execute_plan(selected_plan.steps, profile)
    ↓
MetaStateManager.record_execution()
    ↓
Return: {execution_id, goal, status, steps, strategy, profile, cognition, meta_state, critic}
```
