### 当前架构（Task 25）
```
User Input → Agent._parse_goal()
    ↓
PlannerAgent.plan(goal, tools, memory)
    ↓
plans: [{id:"a", steps:[...]}, {id:"b", steps:[...]}]
    ↓
CriticAgent.evaluate(goal, plans, memory)
    ↓
{selected_plan, score_map, suggestions}
    ↓
ExecutorAgent.execute_plan(selected_plan.steps)
    ↓
record_failure() (on failure)
record_plan_score() (always)
    ↓
Return: {execution_id, goal, steps, replans, critic, memory_summary}
```

### 五 Agent 角色
| Agent | 职责 |
|-------|------|
| Planner | 生成多方案 |
| Critic | 评分 + 选择 + 结构优化建议 |
| Executor | 执行 + 重试 + 自修复 |
| StateManager | 状态持久化 + 失败归因 |
| Memory | 历史经验 + 评分 + 失败模式 |
