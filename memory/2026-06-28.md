## ecomos 项目开发

### 技术栈
- FastAPI + SQLAlchemy + PostgreSQL
- 仓库：https://github.com/qamedeli37-max/Repository-name-ecomos

### 开发流程（固定）
每完成一个 Task：
1. `git add .`
2. `git commit -m "feat: xxxxx"`
3. `git push`
4. 回复格式：
   ```
   Task X 完成
   Git Commit: xxxxxxx
   GitHub：已 Push
   ```

### 完成的 Task
- Task 1-4: Product Center API（4个接口，内存存储）
- Task 5: Repository 仓库层
- Task 6: PostgreSQL + SQLAlchemy 持久化
- Task 7: ProductService 服务层
- Task 8: Supervisor supervisor（POST /execute + intent 识别）
- Task 9: Handler pattern（ProductHandler）
- Task 10: Tool Registry（Tool ABC + ToolRegistry + ProductTools）
- Task 11: Agent class（rule-based intent detection）
- Task 12: OpenAI 集成准备（规则模式兜底，无 API key）
- Task 13: LLM structured JSON output（tool + args 分离）
- Task 14: Multi-step plan execution（plan → loop → results）
- Task 15: Execution trace（execution_id + status per step）
- Task 16: Retry + LLM fix_step（MAX_RETRY=2, auto-heal）
- Task 17: Goal → Plan structure（goal-driven agent）

### 架构演进
```
Task 1-6:   HTTP → API → Service → Repository → PostgreSQL
Task 7-9:   HTTP → Supervisor → Handler → Service → Repository
Task 10:    HTTP → Supervisor → Tool.execute() → Service → Repository
Task 11:    HTTP → Agent (rule) → Tool → Service → Repository
Task 13:    HTTP → Agent (LLM JSON) → Tool → Service → Repository
Task 14:    HTTP → Agent → plan loop → tools → results
Task 15:    HTTP → Agent → traced execution → {execution_id, steps}
Task 16:    HTTP → Agent → execute → retry → fix → trace
Task 17:    HTTP → Agent → goal parsing → planner → executor → trace
```

### 当前架构
```
User Input (string or {goal, constraints, context})
    ↓
Agent._parse_goal()
    ↓
Planner (LLM or rules) → {goal, plan: [{tool, args}, ...]}
    ↓
Executor (retry + fix_step + trace)
    ↓
Return: {execution_id, goal, steps: [{tool, status, result/error}]}
```

### 重要约束
- 不混用 Pydantic models (app/models/) 和 SQLAlchemy ORM (app/db/models/)
- 每个 Task 完成后 git commit + push
- PostgreSQL 密码用 *** 占位
- 没有 OpenAI API key 时 Agent 自动退回到规则模式
- .gitignore: node_modules/, __pycache__/, .env, *.pyc
