
### 当前架构
```
HTTP → API → Service → Repository → SQLAlchemy → PostgreSQL
```

### 项目结构
```
backend/
├── .env                          ← DATABASE_URL
├── requirements.txt
├── app/
│   ├── main.py                   ← FastAPI 入口
│   ├── api/product.py            ← HTTP 接口
│   ├── services/product_service.py ← 业务逻辑
│   ├── repositories/product_repository.py ← 数据存取
│   ├── models/product.py         ← Pydantic Model
│   └── db/
│       ├── database.py           ← Engine + Session + Base
│       ├── init_db.py            ← 建表脚本
│       └── models/product.py     ← SQLAlchemy ORM
```
